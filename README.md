# CodeAlpha Machine Learning Internship Projects

This repository contains the projects completed as part of the CodeAlpha Machine Learning Internship.

## Projects

### 1. Credit Scoring Model

* Built machine learning models to predict credit risk.
* Implemented data preprocessing, feature engineering, and model evaluation.
* Models used:

  * Logistic Regression
  * Decision Tree
  * Random Forest

### 2. Disease Prediction

* Developed a machine learning model for disease prediction using medical data.
* Performed data cleaning, preprocessing, model training, and evaluation.

### 3. Handwritten Character Recognition

* Built a Convolutional Neural Network (CNN) using TensorFlow and Keras.
* Trained on the MNIST dataset to recognize handwritten digits.
* Achieved high classification accuracy on unseen data.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* TensorFlow
* Matplotlib
* Jupyter Notebook
